# RegistrationWorkflow

## Description
Component that renders a succession of screens to register a new user.

## Usage
```tsx
import { RegistrationWorkflow } from '@brightlayer-ui/react-auth-workflow';

const MyComponent = () => {
  return (
    <RegistrationWorkflow>
      {children}
    </RegistrationWorkflow>
  );
};
```
