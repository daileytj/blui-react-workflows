# PasswordTextField

## Description
Component that renders a text field with a visibility toggle.

## Usage
```tsx
import { PasswordTextField } from '@brightlayer-ui/react-auth-workflow';

const MyComponent = () => {
  return (
    <PasswordTextField/>
  );
};
```

## API

- **TextFieldProps** 
    - See [TextField](https://mui.com/components/text-fields/) for more details.
